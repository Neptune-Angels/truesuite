# @truesuite/ui

Shared UI primitives across TrueSuite apps. Empty for now — fill as patterns emerge from `apps/trueqr/components/`.

## Planned exports
- `<Button>`, `<Input>`, `<Card>` — base primitives
- `<Header>`, `<Footer>` — app shells with slot for app-specific nav
- `<MarketingHero>`, `<PricingTable>` — landing-page building blocks

Consumed via `"@truesuite/ui": "workspace:*"` in app package.json.
